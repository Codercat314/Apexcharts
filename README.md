Make sure you have Node package Manager (npm) and node.js

Do a git clone on https://github.com/Codercat314/Apexcharts

  git clone https://github.com/Codercat314/Apexcharts

Switch to the right folder

  cd Apexcharts

To start the project and view the site

  npm run dev

Navigate then to specified site, in this case localhost:5175 

*This project uses vue 3, openmeteoapi and apexcharts
